# avm_lpm

## Description

* Experiment on LPM-AVM: step-1: using Listing Price Model (lpm_v1) to predict listing price, step-2: using listing AVM model to predict sold_price

## Training:
* Step function input:
``` 
{
  "job_key": "TrLpm202201V2",
  "year_month": 202201,
  "task": "training",
  "feature_selection": 1,
  "bucket": "rnd-avm-v2-dev",
  "version": "lpm"
}
```

## Evaluation:

{
